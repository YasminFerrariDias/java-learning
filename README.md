# Java Exercises

Exercícios desenvolvidos no ensino médio durante os primeiros contatos com programação em Java.

## Temas Abordados

- Operações matemáticas e cálculos
- Estruturas condicionais e comparações
- Laços de repetição e vetores
- Médias e classificações escolares
- Cálculos de salário e finanças
- Saúde e dados pessoais
- Cadastros e entrada de dados
- Interface gráfica com JOptionPane

## Como Executar

Requisito: [JDK 11+](https://www.oracle.com/java/technologies/downloads/)

```bash
# Compilar
javac NomeDoArquivo.java

# Executar
java NomeDoArquivo
```

> Arquivos que utilizam `JOptionPane` requerem ambiente com interface gráfica.
